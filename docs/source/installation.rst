Installation
============

.. toctree::
   :maxdepth: 2
   :caption: Contents:

In most cases, you will want to install judge0api from pip. To do this, simply run:

  pip install api

Requirements
------------

The only dependency for judge0api is Python Requests. It will get installed during installation if not already
present.

Python 3.6 (or greater) is required. Lower versions will not work.
